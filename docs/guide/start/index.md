# 开始使用

## 安装

::: code-group

```sh [npm]
  $ npm add vue-virt-grid
```

```sh [pnpm]
  $ pnpm add vue-virt-grid
```

```sh [yarn]
  $ yarn add vue-virt-grid
```

:::

## 依赖

- `"vue": ">=3.0.0"`
